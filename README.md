# AATSEEL-CALLing

Resources for AATSEEL workshop on Russian CALL (Computer-Assisted
Language Learning).

NO RUSSIAN at WorldCALL 5!
Paid resources are marked with a `$`.

Some thoughts about pros and cons of CALL are [here](why_call.md).
